\doxysubsection{Detailed Description}
\doxylink{structcrow_1_1_c_o_r_s_handler}{CORSHandler} is a global middleware for setting CORS headers. 

By default, it sets Access-\/\+Control-\/\+Allow-\/\+Origin/\+Methods/\+Headers to "{}\texorpdfstring{$\ast$}{*}"{}. The default behaviour can be changed with the {\ttfamily \doxylink{structcrow_1_1_c_o_r_s_handler_a02b12344291975cb64215a1d8db52239}{global()}} cors rule. Additional rules for prexies can be added with {\ttfamily \doxylink{structcrow_1_1_c_o_r_s_handler_a432bb066fbb39a476747923dde784eb2}{prefix()}}.
