\doxysubsection{Detailed Description}
A blueprint can be considered a smaller section of a \doxylink{classcrow_1_1_crow}{Crow} app, specifically where the router is conecerned. 

You can use blueprints to assign a common prefix to rules\textquotesingle{} prefix, set custom static and template folders, and set a custom catchall route. You can also assign nest blueprints for maximum Compartmentalization.
